Projects

Optifood

Co-Created a Web Application that allows its user to find recipes based on the ingredients they have in their pantry. With OptiFood, users can register their pantry, and with a system powered by Artificial Inteligence, they can generate recipes giving priority to those that use their soon-to-expire foods, while also reminding them of set dates in order to avoid food waste. The application was created using html, css , javascript, python and OpenAI API.
